# /build - Incremental Implementation

Activate the **Incremental Implementation** skill to build in thin vertical slices with safety gates.

## When to Use
- Ready to write code
- Implementing each task
- When you want feature flags and rollback safety

## What It Does
Enforces:
- **Thin Vertical Slices** - One feature per commit
- **Tests Alongside Code** - Red-Green-Refactor
- **Safe Defaults** - Feature flags and rollback-friendly changes
- **Atomic Commits** - Every commit is deployable

## Key Benefit
Smaller changes = faster feedback = easier rollbacks.

See `.cursor/skills/incremental-implementation/SKILL.md` for full details.
