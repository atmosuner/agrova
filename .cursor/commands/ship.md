# /ship - Shipping & Deployment

Activate the **Shipping & Launch** skill (plus git workflow and CI/CD) for production deployment.

## When to Use
- Ready to deploy to production
- Need a deployment checklist
- Want staged rollouts and rollback procedures

## What It Does
Guides:
- **Pre-Launch Checklist** - Verification before deploy
- **Feature Flag Lifecycle** - Safe feature toggles
- **Staged Rollouts** - Gradual user exposure
- **Rollback Procedures** - Quick recovery if issues
- **Monitoring Setup** - Production alerting

## Key Benefit
Faster deploys are safer deploys. Good processes catch issues before production impact.

See `.cursor/skills/shipping-and-launch/SKILL.md` for full details.
