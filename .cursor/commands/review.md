# /review - Code Review & Quality Gates

Activate the **Code Review & Quality** skill (plus security, performance, and simplification) for pre-merge quality gates.

## When to Use
- Before merging to main
- When you want expert review
- Ensuring code health

## What It Does
Guides:
- **Five-Axis Review** - Style, correctness, complexity, tests, safety
- **Security Check** - OWASP Top 10, auth, secrets
- **Performance Validation** - Core Web Vitals, profiling
- **Simplification** - Reduce complexity while preserving behavior

## Key Benefit
Catch issues before production. Code review is your last quality gate.
